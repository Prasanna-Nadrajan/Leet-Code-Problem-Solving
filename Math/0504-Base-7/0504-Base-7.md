# 504. Base 7

## Problem Description

Given an integer `num`, return a string of its **base 7 representation**.

### Examples

#### Example 1:
**Input:** $\text{num} = 100$
**Output:** $\text{"202"}$

#### Example 2:
**Input:** $\text{num} = -7$
**Output:** $\text{"-10"}$

### Constraints:
- $-10^7 \le \text{num} \le 10^7$
