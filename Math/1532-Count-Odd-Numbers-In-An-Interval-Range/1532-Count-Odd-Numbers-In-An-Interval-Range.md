1523. Count Odd Numbers in an Interval Range

Problem Description

Given two non-negative integers $\text{low}$ and $\text{high}$. Return the count of odd numbers between $\text{low}$ and $\text{high}$ (inclusive).

Examples

Example 1:

Input: $\text{low} = 3, \text{high} = 7$
Output: $3$
Explanation: The odd numbers between 3 and 7 are [3, 5, 7].

Example 2:

Input: $\text{low} = 8, \text{high} = 10$
Output: $1$
Explanation: The odd numbers between 8 and 10 are [9].

Constraints:

$0 \le \text{low} \le \text{high} \le 10^9$
